#### 硬件部分
    PCB板可能画的不是很好，你们可以自行修改或者重新画，这里也上传视频中用的就是视频中用的PCB源文件。
    使用了一段时间后发现电网电压早晚会有波动，出现过几次电压整流滤波后的电源在72V左右，电压有点高，正常220V---48V*1.414=67.8V


#### 安装教程
     **见视频** 
     **DIY这个电源有一点的危险性，建议有一定电子知识基础的朋友DIY。** 
     **一定一定一定一定一定要注意电解电容以及桥式整流不要接反，否则真的会爆炸哦。**
